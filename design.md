# 야놀자 축제 실시간 소통 플랫폼 - 모바일 앱 디자인

## 디자인 철학
- **스타일**: 깔끔하고 모던한 플랫/미니멀 매거진 UI
- **색상 팔레트**: 야놀자 시그니처 블루(#0052FF 계열), 화이트, 세련된 라이트 그레이
- **모션**: Framer Motion을 활용한 부드럽고 매끄러운(Seamless) 페이드/슬라이드 인터랙션
- **레이아웃**: 모바일 포트레이트 방향(9:16), 한 손 사용 최적화

## 스크린 목록

### 1. 홈 스크린 (Home)
**주요 콘텐츠:**
- 상단: 고정 배너 형태의 노른자 긴급 속보 (Marquee 슬라이딩)
- 중단: 노른자 수사대 실시간 현장 뷰 (Masonry/그리드 피드)
- 하단: 크루 광장, 공동 게이지, 제철 계란판

**기능:**
- 실시간 공지사항 자동 스크롤
- 카드 클릭 시 상세 정보 표시
- 도움됨 버튼 클릭 시 색상 변경 애니메이션

### 2. 크루 광장 스크린 (Crew Square)
**주요 콘텐츠:**
- 축제 메이트 구인구직 리스트
- 카셰어링 및 동행 구인 카드

**기능:**
- 리스트 형태의 카드 레이아웃
- 카드 클릭 시 상세 정보 표시
- 필터링 및 검색 기능

### 3. 공동 게이지 스크린 (Shared Gauge)
**주요 콘텐츠:**
- 공동 게이지 프로그레스 바
- 제철 계란판 (활성/비활성 축제)

**기능:**
- 프로그레스 바 애니메이션
- 글래스모피즘 블러 효과

## 주요 사용자 흐름

### 흐름 1: 실시간 속보 확인
1. 홈 스크린 진입
2. 상단 고정 배너에서 긴급 공지 자동 스크롤 확인
3. 필요시 공지 탭하여 상세 정보 확인

### 흐름 2: 현장 정보 확인
1. 홈 스크린 중단부 노른자 수사대 피드 확인
2. 카드 탭하여 상세 정보 확인
3. 도움됨 버튼 클릭하여 피드백 제공

### 흐름 3: 메이트 찾기
1. 크루 광장 탭 이동
2. 구인구직 카드 리스트 확인
3. 관심 카드 탭하여 상세 정보 및 연락처 확인

## 핵심 컴포넌트 스타일

### 헤더 배너
- 배경: 야놀자 블루 그라디언트
- 텍스트: 화이트
- 높이: 56px
- 위치: Sticky (고정)

### 카드 (Masonry/리스트)
- 배경: 화이트
- 테두리: 1px 라이트 그레이
- 모서리: 12px 라운드
- 그림자: 은은한 그림자 (0 2px 8px rgba(0,0,0,0.08))
- 패딩: 16px

### 프로그레스 바
- 배경: 연한 그레이
- 진행 바: 야놀자 블루 그라디언트
- 높이: 8px
- 모서리: 4px 라운드

### 버튼
- 주요 버튼: 야놀자 블루 배경, 화이트 텍스트
- 보조 버튼: 화이트 배경, 블루 텍스트, 블루 테두리
- 패딩: 12px 24px
- 모서리: 8px 라운드

## 애니메이션 가이드

- **페이드 인/아웃**: 250ms, easeInOut
- **슬라이드**: 300ms, easeInOut
- **도움됨 버튼 클릭**: 색상 채우기 애니메이션 (200ms)
- **탭 전환**: 부드러운 페이드 (250ms)
